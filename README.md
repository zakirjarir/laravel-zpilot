# Laravel Installer GUI

A beautiful, premium web-based installer for Laravel applications. Manage your `.env` configuration, run database migrations, and execute seeders with a sleek GUI.

## Features
- ✨ **Modern UI**: Dark mode, glassmorphism, and smooth animations.
- ⚙️ **Environment Manager**: Update `.env` variables easily.
- 🗄️ **Database Setup**: Run migrations and seeders with one click.
- 🚀 **Easy Integration**: Installable via Composer.

## Installation

To install this package from GitHub, add the following to your Laravel project's `composer.json`:

```json
"repositories": [
    {
        "type": "vcs",
        "url": "https://github.com/zakirjarir/laravel-installer"
    }
],
"require": {
    "zakirjarir/laravel-installer": "dev-main"
}
```

Then run:
```bash
composer update
```

## Usage

After installation, the installer will be available at:
`your-app.test/install`
